# v1.0.1 发布记录

> 日期：2026-05-30
> 主题：三平台启动器 + 独立二进制

## 背景

v1.0.0 仅有 `extract_pptx_elements.py` 一个入口，要求用户在终端手动执行 `python3 extract_pptx_elements.py`。对不熟悉命令行的用户门槛过高。

## 改动

1. **macOS 启动器**：新增 `extract_pptx_elements.command`，内嵌完整 Python 源码，双击即可在终端中运行。Intel Mac 无 Python 3 时回退到源码执行。
2. **Windows 启动器**：新增 `extract_pptx_elements.cmd`，内嵌 Python 源码，自动调用系统 Python 执行。
3. **预构建二进制**：`dist/extract_pptx_elements` 为 macOS ARM64 独立可执行文件（PyInstaller 打包）。
4. **构建脚本**：`build_macos.sh` 和 `build_windows.bat` 用于重新打包。
5. **双语 README**：中文在前，English 在后。

## 本次改动的文件

- `extract_pptx_elements.command`：新增 macOS 单文件启动器
- `extract_pptx_elements.cmd`：新增 Windows 启动器
- `dist/extract_pptx_elements`：macOS ARM64 二进制
- `build_macos.sh`、`build_windows.bat`：构建脚本
- `README.md`：双语化

## 修复

- Intel Mac 兼容：`.command` 启动器检测到无 `python3` 时自动回退到 `python`。

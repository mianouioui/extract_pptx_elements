# v1.0.0 发布记录

> 日期：2026-05-30
> 版本：v1.0.0 — 初始发布

## 概述

`extract_pptx_elements` 首次发布。从 PowerPoint `.pptx` 文件中按幻灯片提取图片、视频、音频、图表、图示、嵌入文件和文本。零外部依赖，仅使用 Python 3.8+ 标准库。

## 新增功能

| 类型 | 支持内容 |
|------|----------|
| 图片 | JPG, PNG, GIF, SVG, BMP, EMF, WMF, TIFF, WebP, JFIF |
| 视频 | MP4, AVI, MOV, MKV, WebM, WMV 等 |
| 音频 | MP3, WAV, AAC, M4A, OGG, MIDI, WMA 等 |
| 图表 | 图表 XML、样式和颜色定义 |
| 图示 | SmartArt 图示 XML |
| 嵌入对象 | PDF, DOCX, XLSX, ZIP 等 |
| 幻灯片文本 | 可选 `--with-text` 导出纯文本 |
| 清单 | 自动生成 `manifest.csv` |

### 核心特性

- 幻灯片编号前缀命名（`001_JPG.jpg`, `002_MP4.mp4`）
- 同页同类型重复自动追加 `_02`, `_03` 序号
- `manifest.csv` 完整记录输出文件与 PPTX 内部资源路径映射
- `--media-only` 仅提取图片/视频/音频
- `--overwrite` 覆盖已有输出
- `-o` 自定义输出目录
- 多文件处理，每个文件独立子目录
- 自动扫描当前目录所有非临时 `.pptx` 文件
- 跳过 PowerPoint 临时锁文件（`~$` 前缀）
- Python 3.8+，零第三方依赖

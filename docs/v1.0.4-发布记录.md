# v1.0.4 发布记录

> 日期：2026-05-30
> 主题：中文类型文件夹 + 构建产出重组

## 背景

v1.0.3 之前，提取的文件直接放在输出目录下，按扩展名区分，对中文用户不直观。

## 改动

1. **中文类型文件夹**：提取的文件按类型归入中文子文件夹：
   - `图片/`（images）
   - `视频/`（videos）
   - `音频/`（audio）
   - `图表/`（charts）
   - `图示/`（diagrams）
   - `嵌入文件/`（embedded objects）
   - `文本/`（text）

2. **构建产出重组**：
   - 构建脚本移入 `scripts/`
   - PyInstaller spec 文件放在 `build/specs/`

3. **完成提示优化**：提取完成后明确告知用户输出文件夹位置。

## 本次改动的文件

- `extract_pptx_elements.py`：新增 `KIND_FOLDER_NAMES` 映射表 + `kind_output_dir()` 函数
- `build/specs/`：PyInstaller spec 文件移入
- `scripts/`：构建脚本移入（后续版本已移除）

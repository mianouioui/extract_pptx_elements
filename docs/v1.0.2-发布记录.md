# v1.0.2 发布记录

> 日期：2026-05-30
> 主题：版本号统一 + macOS 传输权限修复

## 背景

1. 三套入口（`.py`、`.command`、`.cmd`）各自硬编码版本号，容易产生不一致。
2. AirDrop/微信/邮件传输后，macOS 会给文件添加隔离标记（`com.apple.quarantine`）并剥离 `+x` 权限，双击时报「无法验证开发者」。

## 改动

1. **版本号变量化**：`.command` 和 `.cmd` 新增 `VERSION` 变量，单点控制版本号。
2. **macOS 自愈机制**：`.command` 启动时自动执行：
   ```bash
   chmod +x "$0" >/dev/null 2>&1 || true
   xattr -d com.apple.quarantine "$0" >/dev/null 2>&1 || true
   ```
   恢复执行权限并移除隔离标记（失败时 `|| true` 不中断脚本）。
3. **清理 README**：移除个人引用。

# v1.2.1 发布记录

> 日期：2026-05-31
> 主题：统一命名——提取器更名为「PPTX 元素提取器」，还原器统一为「PPTX 元素还原器」

## 背景

此前提取器叫「PPTX 内容提取器」，而 1.2.0 新增的还原器叫「PPTX 元素还原器」，两者措辞不统一（「内容」vs「元素」）。本次把两个工具的命名对齐为「元素」系列，README 主标题也合并为"提取 + 还原"的统称。

## 改动

### 中文名称
- 提取器：`PPTX 内容提取器` → `PPTX 元素提取器`
- 还原器：统一为 `PPTX 元素还原器`（启动器代码本就如此，本次补齐 README 标题与正文链接）
- README 主标题：`PPTX 元素提取与还原器`

### 英文名称
- `PPTX Content Extractor` → `PPTX Element Extractor`
- 还原器英文：`Element Restorer` → `PPTX Element Restorer`
- README 主标题英文：`PPTX Element Extractor & Restorer`

### 版本号
- 全部入口（`extract_*` 与 `restore_*` 各 `.py` / `.command` / `.cmd`）升至 1.2.1。

## 改动范围

- `extract_pptx_elements.py` / `.command` / `.cmd`：docstring、argparse 描述、启动器横幅、版本号
- `restore_pptx_elements.py` / `.command` / `.cmd`：版本号（名称本就正确）
- `README.md`：主标题、中英文章节标题与正文锚点链接
- `CHANGELOG.md`：新增 v1.2.1 条目
- `docs/v1.2.1-发布记录.md`：本文档

## 不改动（历史记录）

- `CHANGELOG.md` 过往条目、`docs/` 旧版发布记录里出现的「PPTX 内容提取器」「元素还原器」保持原样——它们是当时版本的事实记录，改了反而失真。

## 验证

- 入口文件（`.py` / `.command` / `.cmd`）已无「内容提取器」与「1.2.0」残留；新名称与 1.2.1 全部到位。
- 四个启动器内嵌 Python 与各自独立 `.py` 逐字节一致；两个 `.cmd` 仍为无 BOM + CRLF。
- 语法检查 + 端到端（提取 → 改图 → 还原，位置不变）通过。

## 后续

- README 法语章节已对齐新命名（`Extracteur d'éléments PPTX`）并补充了还原器章节（`Restaurateur d'éléments PPTX`），与中英文同步。

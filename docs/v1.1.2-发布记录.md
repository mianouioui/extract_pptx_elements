# v1.1.2 发布记录

> 日期：2026-05-30
> 主题：Windows PowerShell 回退 + README 三语化

## 背景

v1.1.1 的 `.cmd` 启动器依赖系统 Python。如果用户 Windows 没有安装 Python，`.cmd` 直接报错退出。

## 改动

1. **PowerShell 回退**：`.cmd` 新增完整 PowerShell 提取逻辑。优先使用 Python 3（`py -3` → `python` → `python3`）；如果 Python 不可用，回退到 Windows 内置 PowerShell 5.1+，利用系统自带 .NET ZIP/XML 组件完成提取。入口不依赖外部 `.py` 或 `.exe`。
2. **README 三语化**：新增完整法语版本（Documentation française），与中文、English 形成三语对称结构。顶部新增多语言跳转链接。
3. **标题更新**：README 主标题改为 `PPTX 内容提取器 / PPTX Content Extractor`。

## 本次改动的文件

- `extract_pptx_elements.cmd`：新增内嵌 PowerShell 回退实现（~800 行）
- `README.md`：三语化 + 标题更新
- `CHANGELOG.md`：新增 v1.1.2 条目

## 技术细节

### PowerShell 回退架构

```
.cmd 启动
├── Python 3 可用？ → 提取内嵌 Python 代码 → 执行
└── Python 不可用 ？ → 提取内嵌 PowerShell 代码 → 执行
```

PowerShell 实现覆盖 Python 版全部功能：图片/视频/音频/图表/图示/嵌入对象提取、文本导出、manifest.csv 生成、CLI 参数解析（`--no-text`、`--media-only`、`--overwrite`、`-o`）。

### 依赖

| 入口 | 要求 |
|------|------|
| `.py` | Python 3.8+ |
| `.command` | macOS + Python 3 |
| `.cmd` | Windows；Python 3 优先，回退 PowerShell 5.1+ |
